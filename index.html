List + Calculator app

This app was created to create a functioning calculator and also have a list in the same page.

I started with a general plan of how I wanted it to look. Then, I created the basic layout using .xml coding in Android Studio.

Once I got the layout how I wanted it, I was able to add .java scripting to make the buttons perform tasks.

There is still more I can do to make the app even better, but this is a great start to getting the functions I wanted.
